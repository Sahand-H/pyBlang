__init__(name, project_path, data=None, blang_args=None, out_path=True, data_name=None, post_process='NoPostProcessor')[source]¶

Constructor for a Blang model object.

Parameters:
  • name – name of the model to be run (name of .bl file)
  • project_path – filepath to the Blang project directory that houses the .bl file
  • data – optional argument to pass in observed data (in .csv format) into blang
  • blang_args – additional runtime arguments, full list of which is here: https://www.stat.ubc.ca/~bouchard/blang/
  • out_path – optional argument, set to false to output to current folder instead of Blang default
  • data_name – optional argument, use if the name of the parameter in the blang model is different to the .csv file’s name
  • post_process – Set post-processing options, defaults to NoPostProcessor, other option is DefaultPostProcessor
